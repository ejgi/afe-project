Privacy Policy

Advanced Forensic Ecosystem (AFE) — All products and services.

⚡ Last Updated: April 2026 // v1.0

01 // Overview

The Advanced Forensic Ecosystem (AFE), which includes the zen-ioc Standalone Platform, the Large Data Explorer VS Code Extension, and their supporting components, is designed from the ground up to be a zero-footprint, privacy-first toolset.

This Privacy Policy describes what data (if any) is collected, why, and how it is handled. Our core design philosophy is simple: the forensic data you analyze belongs exclusively to you.

02 // Data We Do NOT Collect

AFE products are engineered to operate in fully air-gapped, offline environments. As such, we do not collect, transmit, store, or share:

  • Any files, logs, or datasets you analyze using our tools.
  • Any personally identifiable information (PII) such as name, email address, or IP address from end users.
  • Any behavioral analytics, clickstreams, or usage patterns.
  • Any data about the systems or networks you monitor or investigate.
  • Any payment or financial information (all products are currently free).

03 // Optional Anonymous Telemetry

The Large Data Explorer VS Code Extension includes an opt-in anonymous performance telemetry feature. This feature is disabled by default and must be explicitly configured by the user in VS Code Settings (zenExplorer.supabaseUrl).

If and only if you choose to configure and enable this feature, it may log anonymous performance metrics (processing speed, file size ranges) to a user-controlled Supabase instance. No file content is ever transmitted.

04 // Third-Party Services

The AFE Hub portal (this website) is hosted on GitHub Pages. When you visit this site, GitHub may collect standard web server logs (IP address, browser type) as per their own Privacy Statement. AFE has no access to or control over this data.

The VS Code Extension is distributed via the Visual Studio Marketplace. Microsoft's privacy policy applies to the download and installation process.

05 // Local Data Storage

AFE tools create local index files (e.g. .zendx format) on your device to enable O(1) re-loading performance. These files reside exclusively on your local disk and are never transmitted anywhere. You may delete them at any time without affecting original data.

06 // Children's Privacy

AFE products are professional-grade forensic and data analysis tools intended for use by security professionals, IT administrators, and researchers. They are not directed at children under the age of 13. We do not knowingly collect any personal information from children.

07 // Changes to This Policy

This Privacy Policy may be updated periodically to reflect changes in the product suite. The "Last Updated" date at the top of this page will reflect the most recent revision. Continued use of AFE products after any changes constitutes acceptance of the updated policy.
